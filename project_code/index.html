It's like having a jam session with SoundCloud. The idea is to take a series of random sounds from SoundCloud and generate a beat using those as samples.
By combining these sounds, Silly Beat can make new beats by choosing random sounds and combining them using random patterns.
Background pictures are also randomly selected from Tumblr.
